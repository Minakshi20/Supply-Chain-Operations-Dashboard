# Supply-Chain-Operations-Dashboard
📊 Samsung Supply Chain Operations Dashboard

A dynamic and interactive Power BI dashboard designed to analyze and optimize Samsung’s supply chain performance—covering suppliers, inventory, shipments, and customer insights for data-driven decision-making.

📌 Short Description / Purpose

The Samsung Supply Chain Operations Dashboard provides a comprehensive overview of end-to-end supply chain activities. It enables stakeholders to monitor key metrics like revenue, profit, inventory levels, shipment performance, and supplier efficiency—helping improve operational efficiency and reduce delays.

🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Primary tool for building interactive dashboards and visualizations
• 📂 Power Query – Used for data cleaning, transformation, and shaping
• 🧠 DAX (Data Analysis Expressions) – Created KPIs, calculated columns, and measures
• 🔗 Data Modeling – Established relationships between Supplier, Inventory, Shipment, and Customer datasets
• 📁 File Formats – .pbix for development and .png for presentation

📂 Data Source

The dataset represents Samsung supply chain operations, including:

• Supplier data (lead time, performance)
• Inventory data (stock levels, product categories)
• Shipment data (carriers, delays, shipment volume)
• Customer & sales data (platform-wise revenue, order quantity)

The data is structured into multiple related tables enabling cross-filtering and real-time insights.

🚀 Features / Highlights
🔴 Business Problem

Managing a global supply chain involves multiple challenges:

• Delays in shipments affecting customer satisfaction
• Inefficient supplier performance
• Overstocking or understocking inventory
• Lack of centralized visibility into revenue and operations

Answering questions like:

Which supplier has the best lead time?
Which carrier causes the most shipment delays?
Which platform generates the highest revenue?
Are inventory levels optimized?

…is difficult without a unified dashboard.

🎯 Goal of the Dashboard

To build an interactive analytics tool that:

• Provides real-time visibility into supply chain performance
• Helps identify inefficiencies and delays
• Supports decision-making for logistics, procurement, and sales teams
• Optimizes inventory and supplier management
